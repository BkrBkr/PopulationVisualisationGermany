Installation des MapServers

Hinweis: Anleitung für Windows. Es wird angenommen, dass das Laufwerk D:\ existiert und eine Festplatte ist.

1.) Daten kopieren

'mapserver'-Ordner nach D:\ kopieren.

2.) Starten des MapServers in Docker

docker run --restart=always -d -p 5000:80 --mount type=bind,src="d:/mapserver",target=/etc/mapserver/ camptocamp/mapserver:7.0
